# Agente Reactivo Simple - Bartender Virtual
(Sebastian Gonzalez G. - Emanuel Vanegas Castañeda)

Desarrollado para la clase de Inteligencia Artificial, en la Universidad: PJIC.

## Descripción
Este agente simula un bartender virtual que puede:
- Recomendar bebidas según preferencias (dulce, fuerte, refrescante)
- Preparar diferentes cócteles
- Sugerir bebidas según ingredientes disponibles

## Cómo ejecutar
1. Abrir el archivo en Google Colab
2. Ejecutar las celdas en orden: 2-3-4
3. Interactuar con el agente ingresando info

## Comandos disponibles
- `menu`: Ver todas las bebidas disponibles
- `mojito`, `margarita`, etc.: Pedir una bebida específica
- `dulce`, `fuerte`, `refrescante`: Pedir recomendaciones
- `ron`, `tequila`, etc.: Ver qué se puede hacer con un ingrediente
- `salir`: Terminar la interacción
